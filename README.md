# Te cuento un chisme?
En muchas culturas antiguas se creía que el día de cumpleaños era un momento en que los espíritus —tanto buenos como malos— podían acercarse a la persona.
Por eso surgió la tradición de hacer una celebración, encender velas y rodearse de amigos y familiares: se pensaba que la luz y los buenos deseos protegían al cumpleañero de las malas energías.
